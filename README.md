# React Typescript Template

Clone this repository and modify to your needs.

## Purpose

Create React App (CRA) unfortunately comes with a lot of boilerplate code. This makes it hard to adapt to custom needs, for which ones has to eject CRA. This repo serves as a base without CRA while still preserving relevant features.

## Features

To use the template, just run install using your favourite package manager. For example `yarn install`

### Scripts

Use `yarn` or `npm` to run following scripts:

- start: development webpack server for debugging
- build: compile using webpack
- lint: check for code compliance

## Structure

It comes with a blank pages and a proposed folder strucutre for convenience. Feel free to create your own.
